1、 分词：
   
    基于aho-corasick多模匹配找出所有可能的切分路径，将切分路径视为有向无环图，运用Dijkstra算法找出最短分词路径[ O(logn)期望]，
    分词词片权重为极大似然估计的概率（to do)。



2、 标注：
   
    基于隐形马尔科夫模型，运用向前Viterbi算法求最大概率。
    训练语料：199801人民日报（好老）
